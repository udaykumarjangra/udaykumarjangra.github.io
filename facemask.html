Face Mask Detection using Deep Learning

A tool to detect whether a person is wearing a mask or not. Tensorflow in Python used to train the model.

Check this project on Github

The model is trained on 1,376 images and the base model used was the MobileNetV2. As it requires less computation power and can run on small machines. Then few layers are added on top of this model and then the model was trained.
Results on training the model were

  • Training Accuracy - 99.81%
  • Validation Accuracy - 99.64%

The ResNet caffe model was used to detect faces in the live stream which further feed the detected face to the face mask detection model which further gave its predictions.
