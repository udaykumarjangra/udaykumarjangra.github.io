CodeJudge

Code Judge is a Code Judging module with admin panel and user dashboard. Can be installed on any linux server and can be used as a coding competition platform.

  1. Admin Can add, edit and remove problem
  2. New users can register
  3. Python and C++ can be used
  4. All stats on dashboard
  5. Easy to Install
Technology Used:
  1. HTML CSS Bootstrap JavaScript (For Frontend)
  2. PHP for handling forms and connecting to Database
  3. MySQL for database
NOTE: C++ compiler and Python should be installed on the Linux server first before using CodeJudge
Check this project on Github

The problem page for a user contains the list of problems sorted into two sections, if a problem is solved or not.
The user can view the dashboard to check the current standing. On the problem page, the user can select whether to use Python or C++.

The admin dashboard has the list of problems and has options to delete and edit. The admin can add more problems to the database.
Currently it supports only Python and C++ and currently can be installed on only Linux server.
